# RubiksCubeSolver
 
Currently a prototype version of a Rubiks Cube Solver program.

The final program will be able take a video of a Rubiks cube and convert it into a 3D model of the cube. It will then figure out what steps are needed to solve the cube and make these moves on the rendered version of the cube and print out the string of moves so the user can follow along. 

This is just a prototype for how it will work and the final versions will be released soon
